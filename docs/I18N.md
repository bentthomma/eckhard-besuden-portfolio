# Internationalization

## Translation source

`js/i18n.js` contains DE and EN dictionaries in a single IIFE. It exposes `window.i18n` with three methods: `setLanguage(lang)`, `getCurrentLang()`, and `t(key)`.

## HTML attributes

- `data-i18n="key"` -- sets `textContent`
- `data-i18n-placeholder="key"` -- sets `placeholder`
- `data-i18n-aria="key"` -- sets `aria-label`

On language switch, `setLanguage()` walks all elements with these attributes and updates them. It also updates `<html lang>`, `<title>`, and OG meta tags.

## Runtime event

A `langchange` CustomEvent is dispatched on `document` after every switch, with `detail.lang` set to `"de"` or `"en"`.

## Consumers

- **carousel.js** -- listens for `langchange`, re-renders all slides with translated labels via its own `t(key, de, en)` helper that delegates to `Helpers.translate()`.
- **gallery.js** -- uses a `MutationObserver` on the `<html lang>` attribute to refresh overlay labels, detail panel text, and the search placeholder.
- **bid.js** -- builds modal HTML using `Helpers.translate(key, de, en)` at render time.

## Persistence

Language choice is stored in `localStorage("lang")`. On first visit, `navigator.language` is checked; German-locale browsers default to DE, all others to EN.
