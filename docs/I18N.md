# Internationalisierung (DE / EN)

> Quelle: `js/i18n.js`. Bei Widerspruch gilt immer der Live-Code.

## System

- Sprache wird per `data-i18n` Attribut auf HTML-Elementen gesetzt
- `data-i18n-placeholder` für Input-Placeholders
- `data-i18n-aria` für aria-label Übersetzungen
- Sprachwechsel aktualisiert auch `document.title`, `meta[description]`, `og:title`, `og:description`
- `langchange` CustomEvent wird dispatched — scroll.js reagiert darauf (Rebuild Text-Reveals)

## Reservierte Keys (noch nicht in UI)

Diese Keys existieren in i18n.js für zukünftige Features:

- **Lightbox**: lightbox_close, lightbox_prev, lightbox_next, lightbox_of, lightbox_oil, etc.
- **Contact Section**: contact_title, contact_address, contact_phone_label, contact_email_label
- **Cookie Banner**: cookie_text, cookie_accept, cookie_decline, cookie_more
- **Bid Form**: bid_title, bid_name, bid_email, bid_amount, bid_submit, bid_success, bid_error_required
